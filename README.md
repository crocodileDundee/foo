### To initialise the project

## Tests


### To run all tests and rubocop

```bash
bundle exec rspec
```


### To run one file


```bash
bundle exec rspec path/to/test/file.rb
```


### To run one test

```bash
bundle exec rspec path/to/test/file.rb:TESTLINENUMBER
```
